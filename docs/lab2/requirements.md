# Функціональні вимоги AnswerBetter

| ID | Функціональна вимога | Пріоритет |
|---|---|---|
| FR-01 | Система має дозволяти користувачу вводити або вставляти текст повідомлення. | Високий |
| FR-02 | Система має дозволяти користувачу обирати бажаний тон відповіді. | Високий |
| FR-03 | Система має генерувати покращений варіант повідомлення на основі введеного тексту. | Високий |
| FR-04 | Система має відображати результат генерації користувачу. | Високий |
| FR-05 | Система має дозволяти копіювати згенерований текст. | Середній |
| FR-06 | Система має зберігати історію останніх згенерованих відповідей. | Низький |
